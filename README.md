## 🛠️ Technologies Used

- **Frontend**: HTML5, CSS3, Tailwind CSS, Bootstrap
- **Backend**: PHP (basic), MySQL
- **Tools**: Git, GitHub
- **Design**: Responsive layout, Google Fonts, Font Awesome
- **Other**: Contact form with PHP, GitHub Pages-ready structure
